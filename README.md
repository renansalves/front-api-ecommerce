# E-commerce Front-end Project

## Installation

- Passos para instalação do projeto. Versões futuras disponibilizar a opção de subir em um container.

   1. **Clonar o repositorio**

      ```bash
      git clone https://github.com/yourusername/my-ecommerce-project.git
      ```

   2. **Instalar as dependencias com npm**

      ```bash
      npm install
      ```

   3. **Rodar em modo dev**

      ```bash
      npm run dev
      ```

  ## Features

  - **Pagina de listagem de produtos**
    - Mostra a lista de produtos, com imagem, preço e descrição.
    - Permite usuários filtrarem os produtos por categoria ou faixa de preço.

  - **Shopping Cart**
    - Navigate through the product list to add items to the cart.
    - Manage the cart contents: update quantity, remove items, proceed to checkout.
